# Effects, Glass, and Gradients

EdenCSS includes modern visual utilities for polished interfaces.

## Radius

```html
<div class="rounded-sm">Small</div>
<div class="rounded-md">Medium</div>
<div class="rounded-lg">Large</div>
<div class="rounded-xl">Extra large</div>
<div class="rounded-2xl">2XL</div>
<div class="rounded-full">Pill</div>
```

## Shadows

```html
<div class="shadow-sm">Small shadow</div>
<div class="shadow-md">Medium shadow</div>
<div class="shadow-lg">Large shadow</div>
<div class="shadow-glow">Glow shadow</div>
```

## Glass

```html
<div class="glass rounded-xl p-6 shadow-md">
  Glass card
</div>
```

For darker glass surfaces:

```html
<div class="glass-dark rounded-xl p-6 shadow-md">
  Dark glass card
</div>
```

## Gradients

```html
<div class="gradient-primary rounded-xl p-6 text-white">Primary gradient</div>
<div class="gradient-cyber rounded-xl p-6 text-white">Cyber gradient</div>
<div class="gradient-sunset rounded-xl p-6 text-white">Sunset gradient</div>
<div class="gradient-aurora rounded-xl p-6 text-white">Aurora gradient</div>
```

## Gradient text

```html
<h1 class="fluid-title text-gradient">
  Gradient heading
</h1>
```

## Motion

```html
<div class="transition hover-lift">Lift on hover</div>
<div class="transition hover-scale">Scale on hover</div>
<div class="fade-in">Fade in</div>
<div class="slide-up">Slide up</div>
<div class="pop-in">Pop in</div>
```

Motion utilities should be used sparingly. Avoid using too many animations on one page.
